# Mutual Fund Analytics Dashboard

A data analytics project focused on Indian Mutual Fund data.

## Day 1 Deliverables

* Project setup
* ETL pipeline initialization
* Data ingestion of 10 CSV datasets
* Fund master exploration
* AMFI code validation
* Live NAV data fetching using MFAPI
* Data quality reporting

## Technologies

* Python
* Pandas
* NumPy
* SQLAlchemy
* Requests
* Jupyter Notebook
* Plotly
* Git & GitHub
